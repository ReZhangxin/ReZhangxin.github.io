浅拷贝，深拷贝，扁平化

发表于 2017-08-13 | 分类于 JavaScript

数组的浅拷贝

浅拷贝的是对象的指针，而不是对象的本身(这里的对象是Object类型)
如果其中一个发生变化，这个对象也跟着改变

数组的深拷贝

与浅拷贝相反 深拷贝完全拷贝到另一个空间里
修改其中一个对象不会影响另一个对象

数组的扁平化

数组的扁平化，就是将一个嵌套多层的数组 array 转换为只有一层的数组。
[1,[2,[3,4]]] ==>[1,2,3,4]这就是数组的扁平化

1、如果数组都是数字可以用toString()方法
