2、在cmd安装hexo

npm install -g hexo-cli
在安装过程中遇到npm卡顿的情况，解决方法就是修改安装源。
npm config set registry http://registry.cnpmjs.org
npm info underscore如果上面配置正确就会输出一些内容

8、安装git

1) 生成SSH

ssh-keygen -t rsa -C "邮件地址@youremail.com"一路回车密码空就行
一定要注意必须要等待邮件发过来再去进行下一步
（成功的钥匙图标是绿的不是黑的）
然后等待邮件发过来点击进去在桌面的Administrator中找到
.ssh文件夹中的id_rsa.pub 中的秘钥添加到github中的SSH Keys
链接Git $ ssh -T git@github.com
成功提示Hi ReZhangxin! You've successfully authenticated, but GitHub does not provide shell access.

2)配置信息

$ git config –global user.name “ReZhangxin”
$ git config –global user.email “1307881527@qq.com”
没有回答就是好回答说明没有问题

3)部署到Github

npm install hexo-deployer-git --save
